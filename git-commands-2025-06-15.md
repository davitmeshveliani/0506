# 🔧 Git ბრძანებების სია — 15 ივნისი 2025

ეს არის სია ყველა Git ბრძანებისა, რომელიც გამოიყენე დღევანდელი სამუშაოს დროს. თითოეულს ახლავს მოკლე განმარტება:

---

## 📁 ფაილების მართვა
- `git status` ➤ აჩვენებს ცვლილებების სტატუსს
- `git add .` ➤ ყველა ფაილის stage-ში დამატება
- `git add <ფაილი>` ➤ კონკრეტული ფაილის დამატება
- `git commit -m "..."` ➤ ცვლილებების შენახვა commit-ის სახით
- `git branch` ➤ აჩვენებს ლოკალურ ფილიალებს  
- `git branch -r` ➤ დისტანციური ფილიალები  
- `git checkout <branch>` ➤ გადადის ფილიალზე  
- `git merge <branch>` ➤ აერთიანებს ფილიალს  
- `git push -u origin <branch>` ➤ ატვირთავს ფილიალს და აკავშირებს GitHub-თან  
- `git pull origin main` ➤ განაახლებს ლოკალურ ფაილებს GitHub-იდან  
- `git remote -v` ➤ აჩვენებს რემოუტ მისამართებს  
- `git log --oneline` ➤ მოკლე commit-ისტორია  
- `git log --graph --oneline --all` ➤ ვიზუალური გრაფი  
- `git diff` ➤ აჩვენებს ფაილის ცვლილებებს  
- `git stash` ➤ დროებითი შენახვა  
- `git tag v1.0` ➤ ტეგის შექმნა
